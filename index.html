MMAlert

Scrapy JSON Swift Flask AWS HTML CSS JavaScript Python Linux

Built using programmatic Swift for the IOS app and JS, HTML, CSS for the webapp, displaying current or upcoming Mixed Martial Arts (MMA) sporting events. Users can track when each major event occurs and set reminders for the specific time each event begins and have access to a database filled with individual fighter information. Data is scraped from relevant news sources using Scrapy spiders hosted on an ec2 instance that is scheduled to run using AWS lambda and eventbridge. JSON files filled with the scraped data is hosted on a github repo and accessed/distributed by an API Flask server hosted on Heroku.
